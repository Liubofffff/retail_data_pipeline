### Retail Data Pipeline  
Система для автоматической генерации, обработки и хранения данных о продажах в сети магазинов товаров для дома.

### Описание проекта  
Проект представляет собой систему для автоматической генерации, обработки и хранения данных о продажах в сети магазинов товаров для дома. Система эмулирует работу реальной кассовой программы и автоматизирует процесс сбора и анализа данных.

### Структура репозитория  
retail-data-pipeline/  
generate_sales.py - Скрипт генерации CSV-файлов  
read_files.py - Скрипт загрузки данных в БД  
pgdb.py - Класс для работы с PostgreSQL  
sql/ - SQL-скрипт создания таблиц  
config.ini - Конфигурационный файл  
requirements.txt - зависимости проекта  
data/ - Директория для CSV-файлов  
img/ - Скриншоты автоматизации


## Запуск проекта
1. Выполните SQL-скрипт для создания таблицы:  
https://github.com/Liubofffff/retail_data_pipeline/blob/main/sql/createTable.sql

2. Создаем и активируем виртуальное окружение:
- Создаем виртуальное окружение  
python -m venv .venv

- Активируем виртуальное окружение
Для Windows:  
./venv/Scripts/activate
Для Linux/macOS:  
source .venv/bin/activate

- Устанавливаем зависимости  
pip install -r requirements.txt

3. Запускаем скрипт генерации файлов  
python generate_sales.py

4. Запускам скрипт записи данных в БД  
python files_reader.py
